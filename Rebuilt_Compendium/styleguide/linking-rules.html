Linking Rules

This page outlines the internal hyperlinking rules for all profile types within the Eberron HTML Compendium.

1. First Mention = Link

The first time a faction, race, class, subclass, religion, location, or timeline is mentioned on any given page, it must be a direct link to that entity’s profile. Later mentions may remain as plain text.

2. Required Backlinks

  • Subclass → Class
  • Spell → All casting classes and subclasses
  • Ability → Class/Subclass/Source
  • Legendary Item → Basic Item Type

3. Section-Specific Expectations

  • Character Profiles: Link to race, class, subclass, religion, origin, weapon, and notable factions.
  • Class Pages: Link to spell list, subclasses, and notable characters.
  • Subclass Pages: Link to parent class, subclass spells, and abilities.
  • Spell Profiles: Link to relevant casting classes/subclasses with organized lists.
  • Faction & Religion Pages: Link to key NPCs, locations, and other relevant factions.
  • Location Pages: Link to factions, religions, notable NPCs, or events tied to the area.
  • Weapons/Armor/Foci: Legendary profiles must link back to their respective basic item profiles.

4. Timeline Tags

For characters, parties, or locations with multiple timeline versions:

  • Link to the specific era profile (e.g., /sharn-age-of-monsters.html)
  • Or link to the tabbed layout page (e.g., /sharn.html)

5. No Redundant Links

  • Do not link the same term more than once per page section.
  • Do not link to the page itself. For example, the word “longsword” should not be hyperlinked on the longsword profile.
  • Links should not appear in headers unless they function as a backlink.
